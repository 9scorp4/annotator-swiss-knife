# Run Scripts

This directory contains scripts for running the Annotation Toolkit application.

## Files

- `run.bat`: Batch script for running the application on Windows
- `run.sh`: Shell script for running the application on macOS/Linux

## Usage

### On macOS/Linux

```bash
chmod +x run.sh
./run.sh gui  # Launch the GUI
./run.sh --help  # Show all available commands
```

### On Windows

```batch
run.bat gui  # Launch the GUI
run.bat --help  # Show all available commands
```

For more information, see the main README.md file in the root directory.
